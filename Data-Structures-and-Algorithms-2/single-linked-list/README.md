# Single Linked List

## Compile with g++

### Add Node to Middle

```sh
g++ add-node-to-middle.cpp -o add-node-to-middle
```

### Delete Middle Node

```sh
g++ delete-middle-node.cpp -o delete-middle-node
```

## Run Binary

### Add Node to Middle

```sh
./add-node-to-middle
```

Output:
```
Initial linked list:
1 2 3 4 

Final linked list:
1 2 3 5 4
```

### Delete Middle Node

```sh
./delete-middle-node
```

Output:
```
Initial linked list:
1 2 3 4 

Final linked list:
1 2 4
```
